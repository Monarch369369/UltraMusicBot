# UltraMusicBot

این یک ربات تلگرام برای دانلود موزیک با کیفیت 320 و حفظ کاور است.

## راه‌اندازی

1. Python 3 نصب کنید.
2. کتابخانه‌ها را نصب کنید:3. توکن ربات خود را در `bot.py` قرار دهید.
4. ربات را اجرا کنید:## ویژگی‌ها

- دانلود موزیک از تلگرام
- حفظ و تغییر کاور آهنگ
- مدیریت فایل‌ها از طریق دکمه‌ها
